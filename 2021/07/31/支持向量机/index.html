支持向量机

​ 支持向量机(support vector machines, SVM)是一种二分类模型。它的基本模型是定义在特征空间上的间隔最大的线性分类器，间隔最大使它有别于感知机；支持向量机还包括核技巧，使它称为实质上的非线性分类器。支持向量机的策略就是间隔最大化，可以形式化为一个求解凸二次规划的问题，学习算法是求解凸二次规划的最优化算法。支持向量机包括线性可分支持向量机，线性支持向量机，非线性支持向量机。

1 线性可分支持向量机与硬间隔最大化

1.1 线性可分支持向量机

​ 给定一个特征空间上的训练数据集$T = {(x_1, y_1), (x_2, y_2), \cdots,(x_N, y_N)}$，其中$x \in \mathcal{X} = R^n, y_i \in \mathcal{Y} = {+1, -1}, i = 1, 2, \cdots, N$,$y_i = + 1$是正例， $y_i = -1$是负例.

线性可分支持向量机所研究的问题与感知机相同，区别在于感知机是用误分类最小的策略求解，求得的解有无穷多个，线性可分支持向量机利用间隔最大化求最优分离超平面，解只有一个。

1.2 函数间隔和几何间隔

​ 一般来说，一个点距离分离超平面的远近可以表示分类预测的确信程度。在$w \cdot x + b = 0$确定的情况下，$|w \cdot x + b|$能够表示点$x$距离巢萍面对远近。而$w \cdot x + b$的符号与$y$是否一致可以判断分类是否正确，因此，用$y(w \cdot x + b)$表示分类的正确性和准确度，这个就是函数间隔的概念。一般定义超平面关于训练数据集$T$的函数间隔为超平面关于$T$中所有样本点的函数间隔的最小值，也就是$\hat{\gamma} = \underset{i = 1, \cdots, N}{min} \hat{\gamma}$。但是函数间隔受$w$和$b$的影响，只要成比例的改变它们，超平面不变的情况下函数间隔也会跟着改变，因此可以对分离超平面的法向量进行某些约束，比如规范化，使得间隔确定，从而有了集合间隔。进行规范化之后，第$i$个点与超平面$(w, b)$的距离是$\gamma_i = y_i(\frac{w}{||w||} \cdot x_i + \frac{b}{||w||})$,几何间隔也就是训练集中所有样本点的几何间隔的最小值。

1.3 间隔最大化

​ 支持向量机学习的基本想法是求解能够正确划分训练数据集并且几何间隔最大的分离超平面。
